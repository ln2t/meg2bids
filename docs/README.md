## FAQ

### Is meg2bids compatible with other MEG systems?

Currently, meg2bids supports only **Neuromag/Elekta/MEGIN FIF files**. Support for other manufacturers (CTF, BTI/4D, KIT/Yokogawa) may be added in future releases.

### Can I use meg2bids with EEG data?

No, meg2bids is specifically designed for MEG data. For EEG, consider using [dcm2bids](https://github.com/UNFmontreal/Dcm2Bids) or [bidscoin](https://github.com/Donders-Institute/bidscoin).

### How do I report a bug?

Open an issue on [GitHub](https://github.com/ln2t/meg2bids/issues) with:
- Description of the problem
- Steps to reproduce
- Error messages
- Environment details (OS, Python version)

### Where can I ask questions?

Post on [Neurostars](https://neurostars.org/) with the `meg2bids` tag.
